# MySecondBrain 项目规划

## 项目目标
构建一个基于 RAG（检索增强生成）的个人知识库问答系统，支持通过自然语言查询个人笔记。

## 技术路线
- 向量数据库：ChromaDB（本地部署，开源免费）
- Embedding 模型：BAAI/bge-small-zh-v1.5（中文效果好，模型小）
- LLM：DeepSeek Chat（API 调用，性价比高）
- UI 框架：Streamlit（快速搭建 Web 界面）

## 里程碑
1. ✅ 完成基础 RAG 流程（文档加载 → 向量化 → 检索 → 生成）
2. ✅ 完成 Streamlit UI 封装
3. 🔄 支持更多文档格式（PDF、Word）
4. 📅 对话历史和流式输出
5. 📅 多用户支持

## 笔记组织方式
采用 PARA 方法组织知识：
- Projects：正在进行中的项目
- Areas：长期关注的领域
- Resources：参考资料和主题资源
- Archives：已完成或休眠的项
